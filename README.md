# BMI Calculator (React)

A simple Body Mass Index (BMI) calculator built using React and Vite that allows users to input height and weight to determine their BMI and health category.

## 🚀 Features

* Real-time BMI calculation
* Health category classification
* Clean responsive UI
* Built with React + Vite

## 🛠 Tech Stack

* React
* Vite
* JavaScript
* CSS

## ▶️ Run Locally

```bash
npm install
npm run dev
```

## 📦 Build

```bash
npm run build
```
